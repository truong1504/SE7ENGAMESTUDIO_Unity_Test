# SE7EN Game Studio - Unity Intern Test

## Candidate
Nguyen Dinh Truong

## Unity Version
Unity 2022.3 LTS

## Features
- WASD Movement
- Character Rotation
- Top-down Camera Follow
- Kick Ball
- Auto Kick
- Ball Camera Tracking
- Goal Celebration (Confetti)
- Reset Scene

## Controls
- W A S D: Move
- Kick: Kick the nearest ball
- Auto Kick: Automatically kick the farthest ball
- Reset: Reload scene

## Build
Windows build is included in the Build folder.
